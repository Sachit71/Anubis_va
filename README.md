How to install since I found publishing on PYPI real tough, I found an alternative,
Go to your terminal make sure you have git installed,
type the following command ```git clone https://github.com/Sachit71/Anubis_va.git``` 
After that is done type this ```cd Anubis_va```
This should lead you to another Directory named 'Anubis_va'
There type, ```python ANUBIS.py```
And you are good to go!
